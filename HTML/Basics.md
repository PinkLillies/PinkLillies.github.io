# HTML - Basics

## Head, title and body

```html
<!DOCTYPE html>
  <html lang="en">
    <head>
      <title>Hello world!</title>
    <body>
      Hello, world!
    </body>
  </html>
```

## Headings

```html
<h1>Type my Heading 1 here</h1>
<h2>Type my Heading 2 here</h1>
<h3>Type my Heading 1 here</h1>
```

## Ordered list

```html
<ol>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ol>
```

## Unordered list

```html
<ul>
  <li>item 1</li>
  <li>item 2</li>
  <li>item 3</li>
</ul>
```

## Add an image

```html
<img src="cat.jpg">
```

## Add a link

```html
<a href="https://www.google.com">Click Here</a>
```

## Add a form

```html
<form>
  <input type="text" placeholder="Full Name" name="name">
  <input type="submit">
</form>
```


## Add a table

```html
<table>
    <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Will</td>
        <td>Smith</td>
        <td>55</td>
    </tr>
    <tr>
        <td>Eva</td>
        <td>Longoria</td>
        <td>44</td>
    </tr>
<table>
 ```
 
 Output is:
 
 <table>
    <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Will</td>
        <td>Smith</td>
        <td>55</td>
    </tr>
    <tr>
        <td>Eva</td>
        <td>Longoria</td>
        <td>44</td>
    </tr>
<table>
