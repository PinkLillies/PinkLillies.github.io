# CSS - Basics

## Add color

```css
<h1 style="color:blue;">
```
## Align text

```css
<h1 style="text-align:center;">
```


## Or add `<style>` at the top of your file under `<head>` :

```css
<head>
  <style>
    div {
    background-color:orange;
    width:200px;
    height:100px;
    padding:10px;
    margin:20px;
    border:   ;
    font-family:   ;
    font-size:   ;
    font-weight:bold;
    }
  </style>
</head>
```




## Or refer to a CSS file at the top of your file under `<head>` :

```css
<link ref="stylesheet" href="style.css">
```
